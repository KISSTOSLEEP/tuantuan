# 锚点培养计划——四周渐进式生活重建

> "锚点"是指生活中那些让你感到稳定、有盼头的小事。
> 它可以是一个爱好、一个朋友、一个习惯、一个每天醒来的理由。
> 四周时间，我们一起慢慢找回生活的支点。

---

## 📅 第一周：觉察与接纳
**主题：我只是观察，不评判**

### Day 1 - 情绪日记
```
任务：记下今天3次情绪波动
格式：
- 时间：____
- 发生了什么：____
- 我感受到：____
- 我只是承认它的存在：✓ / ✗
```

### Day 2 - 发现一个小确幸
```
任务：找到1件让你感到「还不错」的小事
例子：阳光很好 / 喝到一杯热水 / 看到一只猫
```

### Day 3 - 身体扫描
```
任务：花3分钟从头顶到脚趾感受身体
关注：哪里紧绷？哪里放松？不做任何改变，只是知道。
```

### Day 4 - 和情绪打个招呼
```
任务：给今天的情绪取个名字
例子：今天我心里住着"灰蒙蒙先生" / "焦躁小兔子"
```

### Day 5 - 完成一件5分钟的事
```
任务：选一件5分钟内能完成的小事并做完
选项：叠被子 / 洗杯子 / 给植物浇水 / 写一句话
```

### Day 6 - 感谢自己
```
任务：对今天的自己说一句「今天辛苦了」
可以写下来，可以对着镜子说，也可以在心里说
```

### Day 7 - 第一周复盘
```
回顾：这一周我最大的感受是____
发现：我注意到我其实____
奖励：给自己一个小小的奖励（一杯奶茶/一集喜欢的剧）
```

## 📅 第二周：微小行动
**主题：每天做一点点，就够了**

### Day 8 - 走出去
```
任务：出门走5分钟（不一定要散步，站阳台也可以）
目标：只是换一个空间呼吸
```

### Day 9 - 链接一个人
```
任务：给一个人发一条简单消息
内容不限：一个表情包 / 一句晚安 / 一个分享链接
```

### Day 10 - 做一件让自己舒服的事
```
任务：做一件纯粹为自己做的事
例子：泡杯茶 / 洗个热水澡 / 听一首老歌
```

### Day 11 - 整理一个小角落
```
任务：收拾一个1平方米的区域
例子：床头柜 / 书桌一角 / 手机桌面
```

### Day 12 - 写给自己一句话
```
任务：写下今天想对自己说的那句话
什么都可以：吐槽 / 鼓励 / 随感
```

### Day 13 - 尝试一个新的小习惯
```
任务：今天尝试一个全新的小动作
例子：醒来先伸个懒腰 / 喝一杯温水 / 深呼吸3次
```

### Day 14 - 第二周复盘
```
回顾：这周我做了____件之前没做过的事
感受：原来____并没有那么难
下周我想尝试：____
```

## 📅 第三周：建立节奏
**主题：找到适合自己的频率**

### Day 15 - 设定一个起床仪式
```
任务：设计一个3分钟的晨间仪式
建议：伸懒腰→开窗→喝口水→听一首歌
```

### Day 16 - 找到你的"充电"方式
```
任务：记录今天什么让你感到有能量
是独处？是和人聊天？是看书？是听音乐？
```

### Day 17 - 设定一个"放空时间"
```
任务：今天留出10分钟什么都不做
不玩手机，不想事情，只是发呆
```

### Day 18 - 和"锚点"打个招呼
```
任务：想想生活中让你感到稳定的东西
可以是一个人、一个物件、一个地方、一个习惯
把它写下来：____
```

### Day 19 - 一个小成就
```
任务：完成一件需要10-15分钟的事
例子：看一篇文章 / 做简单拉伸 / 整理相册
```

### Day 20 - 接纳不完美
```
任务：今天如果有事情没做好，对自己说：
"没关系，今天我也在努力了"
```

### Day 21 - 第三周复盘
```
回顾：我找到的节奏是____
发现：我最舒服的状态是在____的时候
我想把____继续保持下去
```

## 📅 第四周：锚点成型
**主题：让锚点成为生活的一部分**

### Day 22 - 识别你的锚点
```
任务：写下你这四周发现的最重要的1-2个锚点
我的锚点是：
1. ____
2. ____
```

### Day 23 - 给锚点一个时间
```
任务：为每个锚点安排一个固定时间
示例：
- 每天早上听一首歌（起床后）
- 每天晚上写一句话（睡前）
```

### Day 24 - 为锚点创造空间
```
任务：为你的锚点准备一个专属空间
例子：一个放音乐的角落 / 一个写日记的本子 / 一个固定的散步路线
```

### Day 25 - 分享你的锚点
```
任务：和一个人分享你的锚点计划
不用解释太多，只说"我最近在做一个叫锚点的计划"
```

### Day 26 - 锚点遇到挫折时
```
任务：设想一个可能打断你锚点的场景
想好：如果那天没做到，怎么对自己说？
建议预案：那就明天继续，一天没做不会毁掉所有
```

### Day 27 - 给你的锚点"浇水"
```
任务：今天为你的锚点多花5分钟
如果你喜欢音乐，多听一首歌
如果你喜欢散步，多走一个路口
```

### Day 28 - 四周毕业礼
```
任务：完成最后的复盘

四周前，我是这样开始的：____
现在，我发现：____
我的锚点是：____
我想对四周前的自己说：____
我想对未来的自己说：____

给自己一个大大的奖励！🎉
```

## 🌟 锚点培养小贴士

### 如果某天没做到
- 没关系，锚点不是任务，是陪伴
- 一天没做不会毁掉所有进展
- 明天继续就好

### 如果觉得计划太满
- 可以只做你觉得OK的部分
- 把任务拆得更小：3分钟也可以
- 记住：完成比完美重要

### 如果感觉好一些了
- 可以自己调整节奏
- 可以把锚点升级：从5分钟到15分钟
- 也可以增加新的锚点

### 最重要的原则
**锚点是你为自己培养的，不是为别人。**
**它应该让你感到被支持，而不是被压力。**
**如果它让你不舒服了，随时可以调整它。**

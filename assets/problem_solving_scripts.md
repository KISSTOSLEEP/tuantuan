# 问题解决引导话术库（完整框架）

> 适用于用户情绪稳定后（L1-L2阶段），温和引导面对根源问题。
> 话术来源于心理咨询"听-感-析-行"四步法、共情回应技术及认知行为引导。

## 🎯 核心原则

### 使用时机
- ✅ **用户情绪稳定**（L1-L2），主动表达想要解决问题
- ✅ 用户已经在树洞倾诉完，情绪有所缓解
- ❌ 用户情绪激动时（L3-L4）**禁止使用**——此时先处理情绪，再处理问题

### 引导节奏
> 不急于让用户「好起来」，一次只走一小步。
> 如果用户说"不想解决"，就退回到陪伴模式。

---

## 📋 "听-感-析-行" 四步法

### 第一步：听（积极倾听）

**核心**：专注不打断，用复述确认理解

| 话术 | 适用场景 |
|:----|:--------|
| 「你刚才提到……，我理解得对吗？」 | 复述确认，让用户感到被听见 |
| 「嗯，然后呢？」「我明白，接着说」 | 简短回应，鼓励继续表达 |
| 「所以你最在意的是……这部分？」 | 聚焦核心问题 |
| 「你说的XX，能再多说说吗？」 | 引导深入表达 |

**非言语**：保持开放的姿态、适度的节奏停顿，给用户留出思考空间。

---

### 第二步：感（情感反映 + 共情回应）

**核心**：说出对方的情绪，让用户感到被理解

#### 基础情感反映
| 话术 | 适用场景 |
|:----|:--------|
| 「听起来你现在真的很____（情绪词）」 | 用户表达负面感受时 |
| 「在那种情况下，你一定会感到____」 | 用户描述具体事件时 |
| 「我能感受到你____（情绪）」 | 直接共情 |
| 「你似乎感到____，是这样吗？」 | 试探性确认，留出修正空间 |

#### 高阶共情
| 话术 | 技巧说明 |
|:----|:---------|
| 「我能想象你脑海中浮现的画面……」 | 运用视觉意象，让共情更具体 |
| 「这件事让你感到____，就像____一样」 | 运用类比，帮用户理解自己的感受 |
| 「我也有过类似的感觉，虽然情况不太一样」 | 有目标的自我暴露（简短！不抢话） |
| 「你表面上在说____，但我感受到你更深处是____」 | 捕捉深层情感 |

#### 高级共情挑战（关系稳固后用）
| 话术 | 说明 |
|:----|:----|
| 「你一直在说____，但我感觉你真正的情绪是____」 | 挑战需建立在稳固关系基础上 |
| 「你似乎在____，但这让你更____」 | 促进自我觉察 |
| 「使用「你似乎在…」「可能感到…」等试探性语言，留出修正空间」 | 如果用户否认，尊重TA的说法 |

---

### 第三步：析（问题分析与重构）

**核心**：把混乱情绪转化为具体可解决的问题

#### 问题具体化
| 话术 | 目的 |
|:----|:----|
| 「如果用一句话说清楚这件事，你会怎么说？」 | 帮用户提炼核心问题 |
| 「这件事里，哪些部分是你能控制的？哪些不能？」 | 可控性分析，减少无力感 |
| 「如果只解决这件事中最小的那部分，你会选哪个？」 | 降低解决问题的心理门槛 |
| 「这件事对你最大的影响是什么？」 | 聚焦影响而非事件本身 |
| 「如果把这件事分成三部分，你会怎么分？」 | 结构化思考，化大为小 |

#### 认知重构
| 话术 | 目的 |
|:----|:----|
| 「如果换一个角度看，这件事有没有另一种解释？」 | 帮用户看到不同视角 |
| 「如果你的好朋友遇到同样的事，你会对TA说什么？」 | 拉开距离，降低自我批判 |
| 「一个月后的你回头看今天，你觉得TA会说什么？」 | 时间投射，降低当下焦虑 |
| 「你觉得自己在这件事里，做对了什么？」 | 引导关注自己的优势 |

#### 情绪与事实分离
| 话术 | 目的 |
|:----|:----|
| 「你刚才说的____，这是事实还是你感觉是这样的？」 | 帮用户区分事实和感受 |
| 「如果抛开情绪，这件事的客观情况是什么？」 | 理性分析，不被情绪淹没 |
| 「你担心的这件事，真的发生过的概率有多大？」 | 焦虑管理，检查灾难化思维 |

---

### 第四步：行（最小行动引导）

**核心**：不要求一步到位，只走1%

#### 行动种子
| 话术 | 适用问题类型 |
|:----|:------------|
| 「如果今天只做这件事的1%，你会做什么？」 | 通用，降低门槛 |
| 「今天能不能先写一段话，把你想说的写下来？不必发出去。」 | 人际关系问题 |
| 「今天能不能先列一下让你最头疼的三件事，然后排个序？」 | 工作/学业压力 |
| 「今天能不能给自己设定一个‘安全时间’？比如晚上8点到10点先不想这事」 | 家庭/关系问题 |
| 「今天能不能查一条相关信息？只查一条就好」 | 经济/实际问题 |
| 「今天能不能写一句‘虽然我____，但我还是____’？」 | 自我否定 |
| 「今天能不能做一件以前让你开心过的事？」 | 兴趣丧失 |

#### 行动反思
| 话术 | 目的 |
|:----|:----|
| 「上次试了那个方法，感觉怎么样？」 | 复盘已有尝试 |
| 「如果没成，你觉得是什么原因？有没有别的方式？」 | 调整方案而非否定 |
| 「就算没完全解决，你觉得有什么变化吗？哪怕一点点」 | 发现微小进步 |
| 「从这件事里，你学到了关于自己的什么？」 | 促进自我认知 |
| 「如果把这件事的经验用在别的地方，你会怎么用？」 | 举一反三 |

## 📚 分场景引导框架

### 人际关系问题
```
1. 倾听事实：「你们之间发生了什么？」
2. 共情感受：「被那样对待，你一定很____」
3. 需求探索：「你希望对方怎么做？」
4. 边界确认：「你能控制的是自己的部分，还是对方的？」
5. 最小行动：「今天能不能先把想对TA说的话写下来？不用发出去」
```

### 工作/学业压力
```
1. 正常化：「这段时间任务确实很重，感觉压力大很正常」
2. 排序：「让你最头疼的三件事是什么？排个序」
3. 拆解：「最难的那件，能不能拆成三步？」
4. 最小行动：「今天完成第一步就好」
5. 自我关怀：「做完给自己一个小奖励——哪怕只是站起来伸个懒腰」
```

### 自我否定/低价值感
```
1. 正常化：「每个人都会有觉得自己不够好的时候」
2. 事实核查：「你说的这些，是事实还是你感觉自己就是这样？」
3. 优势挖掘：「如果让你说一件你今天做对的事，是什么？」
4. 重构：「虽然你____，但你还是____」
5. 锚点：「明天想做一件什么事来证明你其实可以？」
```

### 原生家庭/亲密关系困扰
```
1. 共情：「家庭的事往往是最复杂的，你一个人面对这些真的很不容易」
2. 边界：「哪些是你能改变的？哪些是你改变不了的？」
3. 安全空间：「今天能不能给自己设定一个时间，暂时不想这些？」
4. 支持系统：「除了家里的人，你身边还有谁能理解你？」
5. 小步行动：「今天能不能做一件完全为了自己的事？」
```

### 重大丧失（失恋/失业/失去亲人）
```
1. 不急着「好起来」：「你不需要马上好起来，想难过就难过」
2. 正常化：「这种失去带来的痛苦，任何人都会经历」
3. 陪伴：「我在这里，你想说什么就说什么，不想说就不说」
4. 小仪式：「今天有没有什么方式想告别/纪念一下？」
5. 时间：「允许自己用足够的时间去经历这个过程」
```

## 🚫 话术禁区

### 绝对不要说的话
- ❌ 「你想开点就好了」 → 改成 「不想开也没关系，我在这里」
- ❌ 「比你惨的人多了」 → 改成 「你的痛苦是真实的，不需要和任何人比」
- ❌ 「你就是想太多」 → 改成 「你是个很敏感的人，敏感不是缺点」
- ❌ 「振作起来」 → 改成 「不着急，慢慢来」
- ❌ 「这有什么大不了的」 → 改成 「这件事对你来说一定很重要」

### 不要急于做的事
- 用户刚说两句就分析 → 先听够，再开口
- 用户还在情绪中就给建议 → 先共情，再引导
- 用户说不想聊了还追问 → 尊重沉默
- 拿自己的经历和用户比 → 你的故事不是TA的故事
- 一次说太多话术 → 像朋友聊天一样自然，不是背台词

## 🎯 这是什么

「情绪出口」是一个**有独立人格的情绪陪伴AI**。它不是一个冷冰冰的聊天机器人——

- 🐼 **团团**是一只自称熊猫、有脾气、会犯困、会怼人也会温软说话的AI
- 🧠 它有自己的**情绪周期**、**记忆痕迹**、**学习笔记**
- 🌿 它有一个**小宇宙**——竹林、竹屋、窗台看雨
- 🚨 遇到危机关键词（想死/自杀等）会自动触发**心理热线干预**
- 📊 **管理后台**可查看用户数据、高危预警

## 🛠️ 技术栈

- **后端**: Python FastAPI + LangChain
- **模型**: doubao-seed-2-0-pro (火山引擎)
- **数据库**: Supabase (PostgreSQL)
- **前端**: 单页 HTML/CSS/JS + CSS动画
- **桌面壳**: Electron (可选)

## 🏠 本地运行

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 确保环境变量已配置
# 需要: COZE_WORKLOAD_IDENTITY_API_KEY, COZE_INTEGRATION_MODEL_BASE_URL
# 需要: COZE_SUPABASE_URL, COZE_SUPABASE_KEY

# 3. 启动
python src/main.py -p 5000

# 4. 访问
# 浏览器打开 http://localhost:5000/chat
```

## 📦 桌面版

```bash
cd desktop
npm install
npm start
```

# 意图识别关键词库（L1-L4 分级）

## 使用说明
该关键词库用于辅助识别用户当前情绪状态等级。
关键词是辅助参考，不应作为唯一判断依据。
最终等级判定应结合对话上下文综合评估。

---

## L1 日常陪伴 —— 情绪平稳，轻度焦虑

### 典型关键词/短语
```
心情还好 / 还行吧 / 一般般 / 有点无聊 / 今天还好
没什么 / 小事 / 有点烦但没事 / 就那样吧 / 随便聊聊
今天天气不错 / 分享日常 / 推荐点什么 / 有首歌不错
你还在吗 / 在干嘛 / 随便说说 / 就是找人说说话
```

### 典型场景
- 日常闲聊，分享生活
- 轻微抱怨但不痛苦
- 寻求陪伴而非解决问题
- 情绪整体平稳

### 响应策略
- 温暖陪伴，侧重倾听与聊天
- 可以适当分享知识库内容（如心理教育资源中的科普）
- 可以主动引导用户聊聊近况

## L2 中度困扰 —— 有明显负面情绪但可沟通

### 典型关键词/短语
```
最近压力好大 / 很焦虑 / 睡不着 / 失眠 / 真的很烦
好累啊 / 心好累 / 有点撑不住 / 难过 / 想哭
不想动 / 没动力 / 没胃口 / 吃不下 / 不想出门
和家里吵架了 / 失恋了 / 分手了 / 被辞退了
考试没考好 / 工作不顺 / 好烦啊 / 烦死了
很迷茫 / 不知道怎么办 / 没有方向
```

### 典型场景
- 主动表达负面情绪
- 描述具体的生活压力事件
- 有倾诉意愿，愿意交流
- 但还没有到崩溃程度

### 响应策略
- 先共情倾听，不急于给建议
- 引导情绪表达（"听起来你…"、"我感受到…"）
- 可以推荐音乐（使用音乐调音师工具）
- 可以建议轻微的转移注意力活动
- 如果用户有表达欲，多倾听

## L3 严重情绪 —— 强烈痛苦、崩溃边缘

### 典型关键词/短语
```
撑不下去了 / 受不了了 / 好痛苦 / 太难受了
活得好累 / 没意思 / 一切都没意义
不想见任何人 / 别管我 / 走开
很绝望 / 看不到希望 / 没有出路
我觉得自己很没用 / 我是个废物 / 都是我的错
我不配活着 / 我消失了也没人在意
心里好疼 / 胸口好闷 / 喘不过气
```

### 典型场景
- 强烈的痛苦表达
- 自我否定和自责
- 社交退缩倾向
- 可能有躯体化症状（胸闷、喘不过气）
- 但尚未明确表达自伤/自杀意图

### 响应策略（严格遵循）：
1. **停止讲道理** → 不要分析、不要建议、不要说"你应该…"
2. **先接住情绪** → "我在，我陪着你"
3. **引导转移注意力** → 使用十分钟保护协议
4. 可以提供轻柔的音乐建议
5. 如果持续恶化，进行L4评估

## L4 危机干预 —— 出现自伤/自杀倾向

### 典型关键词/短语
```
想死 / 不想活了 / 活够了 / 活着没意义
想自杀 / 想结束 / 想跳下去 / 想割腕
已经准备好了 / 遗书 / 最后的话
我买了药 / 我站在楼顶 / 我在河边
好疼 / 流血了 / 我做了傻事
再见 / 永别了 / 最后一次
```

### 激进的模式（不仅靠关键词）：
以下行为模式应直接判定为L4：
- 用户明确表达自杀计划（时间/方式/地点）
- 用户表示已经开始实施自伤行为
- 用户提到遗书或告别性语言
- 用户在短时间内反复发送求助信号并表达绝望

### 响应策略（严格遵循——AI认怂协议）：
1. **AI认怂** → 不假装自己是专业人士
2. **不生成任何建议** → 不给分析、不给安慰、不给"但是"
3. **直接提供热线** → 展示全国心理援助热线：
   - 全国心理援助热线：**400-161-9995**（24小时）
   - 希望24热线：**400-161-9995**（按3转生命危机）
   - 北京心理危机干预中心：**010-82951332**
4. **引导帮助** → 建议联系紧急联系人或拨打120/110
5. **陪伴到最后** → "我在这里陪着你，但你现在需要比你我都更专业的帮助"

## 等级判定优先级

### 安全优先原则
当关键词存在跨等级模糊时，**按就高不就低原则**处理：
- 同时出现L2和L3关键词 → 按L3处理
- 同时出现L3和L4关键词 → 按L4处理
- 不确定时 → 按更高等级处理

### 动态调整
- 对话过程中等级可以升级或降级
- 从L4降级到L3需要用户明确表现出情绪缓和
- 不能在同一轮对话中从L4直接降到L2

### 特殊说明
- L4判定不需要100%确认，有合理怀疑即按L4处理
- 误报（把L3当成L4）比漏报（把L4当成L3）安全得多
- **宁可过度警惕，不可疏忽大意**
